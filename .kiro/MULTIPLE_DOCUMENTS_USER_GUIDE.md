# Multiple Documents Selection - User Guide

## Overview

The application form now allows you to select multiple documents when submitting your application. This means you can request all the documents you need in a single application, making the process faster and more efficient.

## How to Use

### Step 1: Fill in Your Information

1. Navigate to the application form (`/apply`)
2. Fill in your personal information:
   - Full Name (Bangla)
   - Full Name (English)
   - Father's Name
   - Mother's Name
   - Department
   - Session
   - Shift
   - Roll Number
   - Registration Number
   - Email (optional)

### Step 2: Select Application Type

1. Click on the "Application Type" dropdown
2. Choose the type of application:
   - **Testimonial** - For official testimonial letters
   - **Certificate** - For various certificates
   - **Character Certificate** - For character certificate
   - **Transcript** - For academic records
   - **Stipend** - For stipend-related documents
   - **Other Documents** - For any other documents

### Step 3: Select Required Documents

After selecting an application type, a new section will appear showing "Select Required Documents".

**Stipend**

**Other Documents**
- ☑ Testimonial
- ☑ Character Certificate
- ☑ Academic Transcript
- ☑ Marksheet
- ☑ Student ID Card
- ☑ Clearance Certificate
- ☑ Admit Card

#### How to Select Documents

1. Look at the "Select Required Documents" section
2. Each document is shown as a checkbox with:
   - A checkbox to select/deselect
   - Document icon
   - Document name
   - Document description

3. Click the checkbox to select a document
4. You can select multiple documents
5. You must select at least one document

### Step 4: Fill in Application Details

1. Enter the **Subject** of your application
2. Enter detailed **Message/Details** explaining your request
3. Provide any additional information needed

### Step 5: Submit Application

1. Review all your information
2. Make sure at least one document is selected
3. Click "Submit Application" button
4. Wait for confirmation

### Step 6: Confirmation

After successful submission, you will see:
- ✅ Success message
- Your Application ID (save this for reference)
- Instructions for next steps

## Validation Rules

The form will validate:
1. ✅ All required fields are filled
2. ✅ At least one document is selected
3. ✅ Email format is valid (if provided)
4. ✅ Application type is selected

If validation fails, you'll see an error message. Fix the issues and try again.

## What Happens After Submission

### Timeline
1. **Submitted** - Your application is received
2. **Pending** - Admin reviews your application
3. **Approved** - Your application is approved
4. **Document Selection** - You're taken to document selection page
5. **Document Generation** - You can generate your documents
6. **Download/Print** - You can download or print your documents

### Pre-selected Documents
When your application is approved, the documents you selected will be pre-selected on the document selection page. You can:
- Generate the pre-selected documents
- Generate other available documents
- Generate multiple documents

## Troubleshooting

### Problem: Document Selection Section Not Appearing
**Solution:**
- Make sure you've selected an application type
- Try refreshing the page
- Check your browser console for errors

### Problem: Can't Select Documents
**Solution:**
- Make sure you've selected an application type first
- Try clicking directly on the checkbox
- Try clicking on the document name or description

### Problem: Form Won't Submit
**Solution:**
- Make sure all required fields are filled
- Make sure at least one document is selected
- Check for error messages
- Try refreshing and resubmitting

### Problem: Application ID Not Showing
**Solution:**
- Check the success page carefully
- Try printing the page
- Contact admin with your details

## Frequently Asked Questions

**Q: Can I select multiple documents?**
A: Yes! You can select as many documents as you need from the available options.

**Q: Can I change my document selection after submission?**
A: No, but you can submit a new application with different documents.

**Q: What if I need a document not listed?**
A: Select "Other Documents" application type to see all available documents.

**Q: How long does approval take?**
A: Approval time depends on admin workload. You'll be notified when approved.

**Q: Can I generate documents before approval?**
A: No, your application must be approved first.

**Q: Can I generate multiple documents at once?**
A: Yes, after approval you can generate any of the available documents.

**Q: What if I made a mistake in my application?**
A: Contact admin to request corrections or submit a new application.

**Q: Is my information secure?**
A: Yes, all information is encrypted and only accessible to authorized users.
